Adding a Domain Name

A domain name is a unique address used to access your website on the internet. Here's how to add a domain name:

  1. Register a domain name through a registrar (e.g., GoDaddy, Namecheap).
  2. Link the domain to your hosting provider by updating DNS records.
  3. Point the domain to the website server where your content is hosted.
  4. Wait for DNS propagation (could take a few hours to a day).
  5. Once DNS changes are complete, your domain name will be live and linked to your website!

Using Notepad++ for HTML Coding

Notepad++ is a powerful code editor that makes HTML coding easier with the following features:

  • Syntax Highlighting: Helps distinguish HTML tags, attributes, and values.
  • Auto-completion: Suggests HTML tag names and attributes as you type.
  • Line Numbering: Helps you track and organize long codes.
  • Multi-tab Editing: Allows you to open and edit multiple files at once.
  • Plugins: Extend functionality, including error checking and code formatting.
  • Light-weight: It's fast and perfect for quick coding tasks.

HTML Code Example Using Basic Tags

Here's an example of how we can use HTML tags:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Example</title>
</head>
<body>
    <h1>This is an Example of HTML Tags</h1>
    <p>This is a paragraph of text using the <code><p></code> tag.</p>
    <ul>
        <li>HTML is simple to learn.</li>
        <li>It is used to build web pages.</li>
    </ul>
    <a href="https://www.w3schools.com">Visit W3Schools</a>
</body>
</html>
